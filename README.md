# US-Labour-Statistics-Excel-Dashboard

# Project Overview

This project was created as part of an assignment at TryCatch Careers, a non-profit organization that helps high school seniors explore potential career paths.

The goal of this project is to build an Excel dashboard that presents labour statistics in a clear and visual format. The dashboard helps communicate employment trends and industry information during school visits to students, parents, and school staff.

The dataset used in this project contains U.S. labour statistics by industry and state from 2017 to 2020.

# Purpose of the Dashboard

The purpose of the dashboard is to present labour market information in a simple and visual way so that it can support discussions about career opportunities with high school students.

The dashboard aims to:

Provide clear visual insights into employment data

Highlight industry trends

Present information in an easy-to-understand format for a non-technical audience

# Dataset Description

The dataset contains labour statistics grouped by year, industry, and state.

Dimensions

Year

Industry

State

Measures

Establishments

Employees

Average Annual Wage

Each record represents labour statistics for a specific industry in a particular state during a given year.
